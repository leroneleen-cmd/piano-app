# 🎹 Clavier — Apprendre le piano en autonomie

Application web pour apprendre le piano, sans installation ni serveur. Le son est synthétisé
avec la Web Audio API et la progression est enregistrée localement dans le navigateur.

**Jouer en ligne :** https://apprendre-piano.netlify.app/

## Fonctionnalités

- **Plan du jour** — routine quotidienne d'objectifs, quota XP ajustable, série de jours (streak), badges à débloquer.
- **Programme** — parcours progressif en 4 semaines (20 jours), objectifs cochables et déblocage séquentiel.
- **Cours** — leçons guidées par niveaux (débutant → avancé) : gammes majeures/mineures, modes, accords, 7es, renversements, arpèges, lecture. Déblocage de niveau en niveau.
- **Découvrir** — clavier jouable 3 octaves (souris + clavier d'ordinateur).
- **Leçons** — reconnaissance des notes.
- **Lecture** — lecture de notes sur la portée (clé de sol et clé de fa).
- **Gammes & Accords** — visualisées et jouées sur le clavier.
- **Chansons** — morceaux en mode guidé.
- **Deux mains** — mode mains séparées (main droite / main gauche / ensemble).
- **Technique** — métronome, exercices de dextérité (doigtés), indépendance des mains, polyrythmes.
- **Théorie** — intervalles à l'oreille, cycle des quintes, valeurs rythmiques.
- **Oreille** — quiz de reconnaissance de notes.

## Clavier d'ordinateur

- Main droite : `A S D F G H J K` (blanches), `W E T Y U` (noires)
- Main gauche (octave grave) : `Z X C V B N`
